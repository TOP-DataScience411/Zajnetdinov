ЭКЗАМЕНАЦИОННОЕ ЗАДАНИЕ 1

  =========================

В задании несколько вариантов (задач). 
Для успешной сдачи экзамена необходимо полностью выполнить любую одну задачу. 
Выполнение дополнительных задач полезно со всех точек зрения, включая повышение баллов за экзамен.

  ==========  1  ==========

Используется набор данных, сохранённый в файле data1.npz

Выполните задачу кластеризации. Количество кластеров известно заранее и равно двум.



  ==========  2  ==========

Используется набор данных, сохранённый в файле data1.npz

Любым способом выделите линейную часть выборки. 
Для этой линейной части выполните задачу регрессии.


  =========================

Код выполнения каждой задачи оформляется в отдельный файл с именем task<номер_задания>.<номер_варианта>.py
При решении нескольких задач разрешено повторно использовать свой же код с помощью импортов.
